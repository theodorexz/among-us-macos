# Homebrew

## What is Homebrew?

Homebrew is a package manager for macOS that allows us to easily install software and manage dependencies. More info can be found on the [official Homebrew website](https://brew.sh/).

## Homebrew Installation

Before we install Homebrew, I would like to mention that Homebrew has two different ways of installation, but in this guide we're gonna be using the terminal, but if needed you can also install Homebrew using the .pkg installer via the [Homebrew repository](https://github.com/Homebrew/brew/releases/latest).

To install Homebrew, open your terminal and paste the following command:

```sh
/bin/bash -c "\$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

Next, add Homebrew to your path:

```sh
echo 'eval "\$(/opt/homebrew/bin/brew shellenv)"' >> ~/.zprofile
eval "\$(/opt/homebrew/bin/brew shellenv)"
```

Lastly, verify Homebrew is installed correctly:

```sh
brew doctor
```
