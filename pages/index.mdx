# Introduction

This is a guide for playing the PC version of Among Us (Steam) on MacOS using [Sikarugir/Kegworks](https://github.com/Sikarugir-App/Sikarugir). As this is the PC version of Among Us we're using, it will require the [Steam version of the game](https://store.steampowered.com/app/945360/Among_Us/). Before you start, I reccemeond you head over to the [Dependencies](./dependencies.mdx) page to be fully prepared for this guide. Additionally, I will make a Epic Games version of this guide, if it's requested.

## Compatibility

Please note this guide was written with modern machines in mind, specifically Silicion chips. This guide has been proven to run multiple versions of macOS, spanning from macOS 26.5 to macOS 27 Developer Beta, on a M4 chip. I don't expect much differences in the experience from any of the Silicon chips or from the software versions as they all are efficent and Among Us is far from graphically intenseive for the Silicon chips, although if you are experiencing problems please submit an issue via the [GitHub Repository](https://github.com/canbegold-yt/among-us-macos), make sure to include at minimum your software version, build number, chip type (example A18 Pro, M4 Max, M2), and other details.

## Where to Start

First, visit the [Dependencies](./dependencies.mdx) page to ensure you have all the necessary software installed. Once you have everything set up, you can proceed to the [Guide](./guide.mdx) page for step-by-step instructions on how to get Among Us running on your MacOS device. After following the guide, you should be able to smoothly run Among Us on your Mac without any issues. If you encounter any problems, refer to the [FAQ & Common Issues](./faq-and-common-issues.mdx) page or reach out for support through the GitHub repository.
