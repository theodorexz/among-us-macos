# Rosetta 2

## What is Rosetta 2?

Rosetta 2 is a translation layer that allows you to run Intel-based applications on Apple Silicon Macs. This is quite a boring topic in my opinion, but if you're interested then feel free to take a look at the [official Apple document](https://support.apple.com/en-us/102527). Rosetta 2 is planned to be deprecated and no longer usable past macOS 28, but by then there will possibly be a solution. For now it works!

## Rosetta 2 Installation

Rosetta 2 is installed automatically on your Mac when you try to open an Intel-based app for the first time. But, for our guide, we will be installing it manually because it bypasses the annoying password prompt and incase of if you launch Steam later without internet it won't be downloaded, so why not do it manually. Anyways, first we need to open our terminal (zsh) and run the following command:

```zsh
/usr/sbin/softwareupdate --install-rosetta --agree-to-license
```
