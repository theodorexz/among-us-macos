# Sikarugir [Homebrew Required]

## What is Sikarugir?

Sikarugir is a wrapper. If you're don't know what a wrapper is, it is essentially an isolated environment for a certain application, like in our case Steam. The wrapper translates the Windows code into code that your Mac system can read (vice versa).

More info on this topic can be found on the [official Sikarugir repository located on GitHub](https://github.com/Sikarugir-App/Sikarugir).

## Sikarugir Installation

Since Sikarugir is distributed through Homebrew, our first step is to open the terminal and run these commands together to install Sikarugir.

```zsh
brew trust Sikarugir-App/sikarugir
brew install --cask Sikarugir-App/sikarugir/sikarugir
```
