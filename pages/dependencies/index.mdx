# Dependencies

For our initial setup, we'll need to install a few dependencies. These dependencies will help us manage the project efficently, and save time over the course of this guide. All of these installations are done in the terminal. Make sure to install the following dependencies in order:

1. [**Homebrew**](./dependencies/homebrew.mdx) - Homebrew is a package manager for macOS that allows you to easily install software and manage dependencies. Homebrew can be installed two different ways.

2. [**Rosetta 2**](./dependencies/rosetta2.mdx) - Rosetta 2 is a translation layer that allows you to run Intel-based applications on Apple Silicon Macs. Rosetta 2 can be installed manually or automatically.

3. [**Sikarugir**](./dependencies/sikarugir.mdx) - Sikarugir is a wrapper. If you're don't know what a wrapper is, it is essentially an isolated environment for a certain application, like in our case Steam. The wrapper translates the Windows code into code that your Mac system can read (vice versa). Sikarugir can be installed through Homebrew in the terminal.
